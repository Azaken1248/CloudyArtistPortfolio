# Cloudy Portfolio Frontend Docs

This folder contains the planning and architecture notes for the frontend-only portfolio experience. The admin site is intentionally out of scope here and should be documented separately.

## Source of Truth

- The provided palette and mood-board images define the visual direction.
- The typography reference image is the basis for the headline and body font pairing.
- The frontend should stay faithful to the reference mood while still being practical, responsive, and readable.

## Scope Notes

- React + TypeScript is the target frontend stack.
- Tailwind CSS is the styling system for implementation.
- No code should be added yet; this folder is for planning only.
- Admin dashboard behavior, auth, and backend APIs are excluded from this phase.
