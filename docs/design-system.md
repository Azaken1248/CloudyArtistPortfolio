# Design System

## Goal

Create a premium portfolio interface that feels airy, editorial, and image-forward while remaining fast and easy to use on mobile.

## Visual References

The provided images suggest three important directions:

- A soft, translucent layered layout with rounded containers and blurred panels.
- A cool, calm palette built around blue, white, and neutral gray tones.
- A serif headline treatment that feels elegant and artistic, paired with a clean sans-serif UI font.

These references should be treated as the visual north star for the frontend.

## Color Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Primary | #2D7DFF | Primary buttons, active navigation, selection accents, progress states |
| Secondary | #E0F2FF | Soft backgrounds, subtle tints, glass panels, hover surfaces |
| Tertiary | #FFFFFF | Main page background, cards, content containers |
| Neutral | #4A5568 | Body text, muted labels, inactive UI, image hover dimming |

## Typography

- Headlines: Libre Baskerville
- Body and labels: Space Grotesk

### Typography Rules

- Use the serif font for large section titles, hero statements, and atmospheric accent headings.
- Use the sans-serif font for navigation, buttons, form labels, descriptive text, and metadata.
- Keep headline sizing dramatic but controlled so the page still reads cleanly on smaller screens.

## Layout Language

- Use a white or near-white base with blue-tinted secondary surfaces.
- Use layered cards instead of dense boxed sections.
- Allow oversized typography to overlap or sit behind content in a restrained way.
- Keep the admin site separate so the portfolio can stay visually focused and uncluttered.

## Interaction Language

- Primary blue is reserved for meaningful actions and active states only.
- Hover states should feel subtle, with restrained dimming and movement.
- Translucent navigation should use blur and low-opacity tinting for depth.
- Motion should support hierarchy, not decorate every element.

## Reference Interpretation

The reference boards point toward a calm, refined portfolio rather than a loud marketing site. The overall feel should be soft, curated, and editorial, with the artwork itself doing most of the visual work.
